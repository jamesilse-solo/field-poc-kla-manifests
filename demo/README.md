# KLA Demo — Quick Start

**Demo date:** April 23, 2026 — 12:00pm PT  
**Presenter:** James Ilse  
**Cluster:** `kla-agentic-cluster` (EKS, us-west-2)

## Prerequisites

```bash
aws sso login
aws eks update-kubeconfig --region us-west-2 --name kla-agentic-cluster
export OPENAI_API_KEY="<your-key>"   # required for OpenAI backend
```

## Run order

```bash
# 1. Validate environment
bash demo/scripts/00-preflight.sh

# 2. Load tokens and gateway URL
source demo/.env

# 3. Deploy demo additions (Dex, OpenAI, docs MCP, AgentRegistry)
bash demo/scripts/01-setup.sh

# 4. Open UIs (see demo/ui/port-forwards.md)

# 5. Run themes in order
bash demo/scripts/02-mcp-workflows.sh   # Theme 1: MCP Federation
bash demo/scripts/03-identity-auth.sh   # Theme 2: Identity & Auth
bash demo/scripts/04-governance-ops.sh  # Theme 3: Governance & Ops

# Teardown after demo
bash demo/scripts/99-teardown.sh
```

## What this demo adds (on top of existing base setup)

| Addition | Namespace | Purpose |
|----------|-----------|---------|
| Dex IdP | `dex` | OIDC identity broker (simulates Entra ID) |
| JWT auth policy (Dex) | `agentgateway-system` | Replaces Keycloak JWKS with inline demo keys |
| search.solo.io MCP | `agentgateway-system` | Remote MCP backend at /mcp/docs |
| OpenAI backend | `agentgateway-system` | GPT-4o at /openai (requires OPENAI_API_KEY) |
| AgentRegistry | `agentregistry` | Agent + skill catalog, chart v0.2.1 |

## Base setup (already deployed via ArgoCD)

- AgentGateway Enterprise v2.3.0-rc.1
- Bedrock backends: Haiku, Mistral, LLaMA3 at `/bedrock/*`
- Mock vLLM at `/mock`
- MCP website-fetcher at `/mcp`
- Guardrails (prompt injection, PII, credentials)
- Tiered rate limiting (free/standard/premium/admin)
- cert-manager TLS
- Gloo UI, Grafana
